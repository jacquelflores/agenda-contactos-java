# 📒 Agenda de Contactos en Java

Aplicación de consola en Java para gestionar contactos usando **HashMap** y **Programación Orientada a Objetos (POO)**.  
CRUD básico: agregar, listar, buscar y eliminar contactos.  

---

## 🛠️ Tecnologías
- Java ☕
- HashMap 🗂️
- Programación Orientada a Objetos (POO) 👩‍💻
- IDE: IntelliJ 💻

---

## ✨ Características
- ➕ Agregar un contacto con nombre, teléfono y email
- 📋 Listar todos los contactos
- 🔍 Buscar un contacto por nombre
- ❌ Eliminar un contacto
- ⚠️ Manejo de duplicados

---

## 🎓 Aprendizajes
- 💡 Uso de Map para almacenamiento en memoria
- 🏗️ Separación de responsabilidades (model, service, main)
- 🖥️ CRUD básico en consola
- 📝 Buenas prácticas en commits y estructura de proyecto
